# react-native-easy-uuid

Library to provide native uuids. 2.0.0+ supports TurboModules/New Arch only.

## Installation

```sh
npm install react-native-easy-uuid
```

## Usage


```js
import { uuid } from 'react-native-easy-uuid';

// ...

const result = uuid();
```
